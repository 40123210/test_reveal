隨身可攜程式系統的功能

1. 隨身碟或隨身硬碟帶著系統設定好了，到任何電腦都可不必安裝，隨插隨用，除了方便快速省時之外，每個人的環境都一樣，在協同時可以共同使用。

2. Anaconda 3 的 Python 免費發佈版本，提供各種工程與科學運算模組套件。

3. Jupyter 在協同時可以多人共用工程運算平台，可以寫筆記甚至可以做成網誌文章。提供 Jupyter，啟動後即可自動辨識電腦所在的 IP 位址設定，自動開啟 IPython Notebook 功能。

4. 提供 SciTE 文字編輯器，經由 python.properties 設定，可以直接在編輯器中執行簡單的 Python 3 程式。

5. 文字檔不易把多媒體資料整合一起，就網誌來說會比較簡單。使用 Leo Editor 寫比較簡單的內容。

6. 提供 Git client 端工具

7. 提供 Solvespace 3D 參數繪圖程式
