指令及語法

coding : 編碼

import : 導入

random : 亂數

print : 列印內容

input : 取得使用者輸入

int(integer) : 整數

len : 算 list 長度

str(string) : 字串

range : 數值序列

float : 浮點數

for : 迴圈

while : 迴圈 當

if : 判斷式 假如

# : 單行註解

''' : 多行註解

<= : 小於或等於

< : 小於

> : 大於

== : 等於

!= : 不等於
