使用 GIT

將近端網站資料送到遠端。

利用 git 新增所改變的程式或內容檔案。

1. git clone

加上網址，即可下載內容

2. git add .

增加它的改變 把它的改變加到它的資料庫裡面 ，是可以逐一列出來，但我們初學，不必全都知道。

3. git commit -m "yy"

我們到底做了什麼事情，要做一個標記，-m 就是注解，yy就是你註記的內容。

將暫存區中的登陸內容送到遠端資料倉儲 (remote repository)。

4. git push

當其他使用者利用 git clone 時，就會下載遠端資料倉儲中最新的更新版本。

5. 設定使用者

git config user.email 設定使用者帳號。

git config user.name 設定使用者名稱。

6. git branch

列出目前有多少，亦可確認所在分支位置。

加上 gh-pages 即可建立gh-pages分支。

7. git checkout

切換分支，加上該分支名稱即可切換到該分支。

download 與 git clone 差別

倉儲如有2個以上分支使用 download 的話，只會下載該分支，而使用 git clone 便會將所有分支下載下來。
