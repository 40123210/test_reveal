Git，Github 與 Github Pages

Git

Git 是一組可以讓使用者在近端電腦上執行，且主要目的是用來進行文本資料的分散式版次管理之用。Git 系統包含近端的 Git 指令工具與遠端的 Git 伺服器。

Github

Github 是一套提供分散式版次管理的知名 Git 伺服器，使用者可以透過 https://github.com/ 網站，自行登記使用帳號。

請參考 Github

Github Pages

Github Pages 則是 Github 伺服器上所提供的一種服務，可以讓使用者將超文件格式的資料，利用特定分支名稱的設定，將最新版的超文件內容呈現在 github.io 網站上。
