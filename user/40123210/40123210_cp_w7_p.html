print函式列印字串

用雙引號或單引號或三引號圈起的內容就是字串

利用print列印單一 * → print("*")

利用print列印三個 * → print("***")

利用print列印空白與 * → >print(" *")

利用乘號print列印五個 * → print("*"*5)

for迴圈

透過iterator逐一執行迴圈內容

→ for i in range(5):

print(i)

→ for i in range(1, 5):

print(i)

只列印不等於3的數值

→ for i in range(1, 5):

if i != 3:

print(i)

只列印用2可以整除的偶數

→ for i in range(1, 5):

if i %2 == 0:

print(i)

只列印大於3數值

→ for i in range(1, 5):

if i > 3:

print(i)

只列印大於3或者小於2的數值

→ for i in range(1, 5):

if i > 3 or i < 2:

print(i)

只列印大於2且小於5的數值

→ for i in range(1, 5):

if i < 5 and i > 2:

print(i)

利用help(print)檢視print函試用法

→ for i in range(1, 5):

print(" "*(i-1), end="")

print("*")
