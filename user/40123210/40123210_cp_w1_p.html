為什麼要選用 Python？

有關作業任務一的說明

Python 的極簡主義讓你可以專注在解決問題上，而不是花時間在了解機器細節、打字或迷失於語言的各種功能上。再來就是 Python 的社群很健全，從科學、網站到硬體都可以找到相應的函數庫，可以幫你節省很多時間。

語法自然而易於初學者學習。

誰適合使用Python?

由於Python簡單學習，對於程式語言不熟悉的人可由此開始，如初學者、學生、對語法有興趣的人等等，皆可由此入門。

心得結論

第一次學的語言是C++，對我來說是很困難的，很多語法都聽不懂也不會使用，回來重修後從Python學起，一切變得很容易，雖然沒有到熟練，但比起剛初學習C++更加容易上手。
